Travelio is a modern, original travel & tour WordPress theme with a warm orange + navy palette.
It includes custom post types for Tour Packages and Destinations, a full homepage layout
(hero search, destinations grid, tour packages, testimonials), and integration hooks for the
free WP Travel Engine plugin.

== Installation ==

1. Zip the entire `travelio` folder so that `style.css` is at the root of the zip
   (the zip should contain `travelio/style.css`, `travelio/functions.php`, etc).
2. In WordPress, go to Appearance → Themes → Add New → Upload Theme.
3. Upload the zip, then click Activate.
4. Go to Appearance → Customize to set your hero text, contact details and social links.
5. Add your tours under Tours → Add New and destinations under Destinations → Add New.

== Recommended plugins ==

* WP Travel Engine — for full booking, payments, and trip management
  (the theme detects it automatically and renders booking forms accordingly).
* Contact Form 7 or WPForms Lite — for the contact page form.

== Customization ==

* Colors and fonts live in `style.css` under the `:root` block — change the CSS variables
  there to re-skin the whole theme.
* Homepage sections are in `front-page.php` — duplicate or remove sections freely.
* Menus: Appearance → Menus → assign to "Primary Menu" and "Footer Menu".
