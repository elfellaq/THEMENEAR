# Travelio - Modern Travel & Tour WordPress Theme

A professional travel theme inspired by Travila, featuring full WP Travel Engine integration.

## Features

- 🌍 **Travila-style Homepage** with advanced search and filters
- ✈️ **WP Travel Engine Integration** (FREE) for booking functionality
- 📅 **Check-in/Check-out Date Pickers** in search form
- 🛒 **Checkout in Trips** button for easy booking
- 🎨 **6 Demo Tours** ready to import (Paris, Tokyo, NYC, Bali, Rome, Dubai)
- 🏝️ **6 Destinations** with beautiful imagery
- 📱 **Fully Responsive** design
- ⚡ **Fast & Lightweight** code structure

## Quick Start

1. Install theme via WordPress Admin → Appearance → Themes
2. Install **WP Travel Engine** plugin (recommended)
3. Go to **Appearance → Import Demo Content** to populate your site
4. Start customizing!

## Demo Content Importer

The theme includes a built-in demo importer that adds:
- 6 fully configured tours with images, prices, and itineraries
- 6 popular destinations
- Essential pages (About, Contact, FAQ)
- Navigation menu setup

Access it at: **WordPress Admin → Appearance → Import Demo Content**
