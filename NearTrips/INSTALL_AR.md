## 🚀 خطوات التثبيت (5 دقائق):

### 1️⃣ رفع الثيم
```
WordPress → Appearance → Themes → Add New → Upload Theme
اختر ملف: NearTrips-Theme.tar.gz
```

### 2️⃣ تنصيب WP Travel Engine
```
Plugins → Add New → ابحث عن "WP Travel Engine"
Install → Activate
```

### 3️⃣ استيراد المحتوى التجريبي
```
Appearance → Import Demo
اضغط: 🚀 Import Demo Content Now
```

### 4️⃣ إعداد الصفحة الرئيسية
```
Settings → Reading
A static page → Home → Save
```

### 5️⃣ حفظ الروابط
```
Settings → Permalinks → Save Changes
```
